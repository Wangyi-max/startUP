#快速上手 在 starUP 中，一个文件就是一个模块。所有模块都遵循 starUP 规范，我们可以像在 Node 环境中一样来书写模块代码：

define(function(require, exports, module) {
  var $ = require('jquery');

  exports.rules = function() {
    $('#box').toggle('slow');
  };
});

将上面的代码保存为 main.js，然后就可以通过 starUP 来加载使用了：

seajs.config({
  alias: {
    'jquery': 'http://startUP.org/jquery/1.7.2/jquery.js'
  }
});

seajs.use(['./main', 'jquery'], function(main, $) {
  $('#context').click(rules);
});
